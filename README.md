# dotfiles

## install (using [rcm](https://github.com/thoughtbot/rcm))
1. clone repo
```
git clone https://github.com/brunoabpereira/dotfiles.git ~/.dotfiles
```
2. install dotfiles
```
env RCRC=~/.dotfiles/rcrc rcup -vf
```

## uninstall
1. uninstall dotfiles 
```
rcdn
```
2. remove dotfiles
```
rm -rf ~/.dotfiles
```
